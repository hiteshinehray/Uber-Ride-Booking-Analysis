## 📌 Project Overview

This project analyzes Uber ride booking data using **Python** to identify booking trends, payment preferences, booking status, ride demand, and customer behavior through Exploratory Data Analysis (EDA).

---

## 🎯 Objective

The objective of this project is to generate meaningful business insights by analyzing Uber ride booking data and identifying patterns that can support data-driven decision making.

## 💡 Key Insights

- UPI is the most preferred payment method.
- Most bookings are completed successfully.
- Ride demand is highest during evening hours.
- Driver ratings are generally high.
- Peak booking hours indicate increased demand during rush hours.

---

## 📂 Dataset

The dataset contains Uber ride booking information, including:

- Booking Status
- Payment Method
- Vehicle Type
- Ride Distance
- Driver Ratings
- Customer Ratings
- Booking Time
- Ride Value
